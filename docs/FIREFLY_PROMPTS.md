# Adobe Firefly Image Generation Prompts for MetricAlgo

This document contains all the Adobe Firefly prompts needed to generate images for the MetricAlgo website redesign.

## General Style Guidelines

**Consistent mood across all images:**
- Dark, sophisticated backgrounds with soft gradients
- Metallic or glass textures
- Electric cyan (#00D9FF) and platinum (#E5E7EB) highlights
- Subtle glows and light effects
- Premium, futuristic aesthetic
- High contrast but refined

**Technical specifications:**
- Output format: PNG (high resolution)
- Aspect ratio: Varies by section (specified below)
- Compression: Optimize for web without losing quality

## Notes for Designer

- Start with HIGH priority images first
- Maintain consistent color palette: Electric cyan (#00D9FF), Platinum (#E5E7EB), Graphite/Dark blues
- All images should have a premium, sophisticated feel
- Avoid overly complex or busy compositions
- Focus on clarity and immediate visual understanding
- Test images against dark backgrounds (#0A1120, #121A24)
- Ensure sufficient contrast for text overlay where needed
- Generate at 2x resolution for retina displays, then optimize for web
